下载并更改 hexo 默认主题🎈

前言

以 Hexo框架搭建的站点有两份主要的配置文件，其名称都是 _config.yml 。 其中，一份位于站点根目录下，主要包含 Hexo 本身的配置；另一份位于主题目录下，这份配置由主题作者提供，主要用于配置主题相关的选项。
一般我们将前者称为“站点配置文件”， 后者称为“主题配置文件”。
